# Developer Life Archive

Developer Life Archive is a planned fullstack devlog portfolio for showcasing projects, case studies, technical growth, and developer notes in one place.

## Planned Features

- Project showcase
- Project detail case studies
- Dev journey timeline
- Tech stack showcase
- Devlog/blog
- Contact and CV section

## Tech Stack

- Backend: Java 21, Spring Boot 3.5, PostgreSQL, Spring Data JPA
- Frontend planned: React, Vite, TailwindCSS

## Project Structure

- `/backend` - Spring Boot backend API
- `/frontend` - Planned React frontend
- `/database` - Planned database scripts and notes
- `/docs` - Architecture, roadmap, API, database, UI, and deployment notes

- Backend skeleton initialized
